三步开始使用

不到一分钟，即可开始用自然语言管理店铺

1

安装 CLI

npm install -g @optima-chat/optima-cli
2

登录账号

optima auth login

浏览器会自动打开，完成 OAuth 授权

3

开始对话

在 Claude Code 中直接用自然语言说话：

"创建商品..."
"查看订单..."
"库存低于 5 的商品"

需要 Node.js >= 18
